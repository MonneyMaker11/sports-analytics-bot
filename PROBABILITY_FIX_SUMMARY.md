# Исправления вероятностей для матчей сборных (Март 2026)

## Проблемы

### 1. Неверные коэффициенты
**Было:** Кф на Боснию ~1.9 (неверно)  
**Стало:** Кф на Боснию ~7.0-8.5 (реалистично)

### 2. Нереалистичные вероятности
**Было:** Босния 52% победы (абсурдно)  
**Стало:** Босния 17% победы (реалистично)

## Решения

### 1. Исправлено извлечение коэффициентов (`MarketAnalyzer._extract_odd`)
- Добавлена поддержка рынка "Match Winner"
- Обработка нескольких букмекеров
- Маппинг label'ов: "Home"/"Draw"/"Away" → "1"/"X"/"2"

### 2. Добавлен рейтинг FIFA (`MarketAnalyzer.FIFA_RANKINGS`)
- 60+ национальных сборных с актуальными рейтингами
- Италия: #7, Босния: #45
- Используется для корректировки вероятностей

### 3. Улучшен расчет для сборных (`EnhancedPoissonCalculator`)
- **Снижено домашнее преимущество:** 1.06 вместо 1.12-1.15
- **FIFA ranking adjustment:** множители на основе рейтинга
  - Топ-10: 1.20-1.30
  - 11-25: 1.05-1.19
  - 26-50: 0.90-1.04
  - 50+: 0.75-0.89
- **Tighter caps:** ограничены expected goals для сборных

### 4. Market Odds Blending
**Когда коэффициенты доступны:**
- 40% модель + 60% рынок (для сборных)
- 60% модель + 40% рынок (для клубов)

**Когда коэффициентов нет:**
- 20% модель + 80% FIFA ranking (для сборных)
- Формула на основе strength ratio

## Как это работает

### Для матчей сборных с коэффициентами:
```
1. Извлекаем кф из API-Football (Betano, Superbet, etc.)
2. Рассчитываем market implied probabilities (с удалением vig)
3. Blend: 40% Poisson model + 60% market odds
4. Результат: реалистичные вероятности
```

### Для матчей сборных без коэффициентов:
```
1. Берем FIFA ranking команд
2. Рассчитываем strength: 100/ranking
3. Добавляем home advantage (1.06)
4. Рассчитываем ranking-based probabilities
5. Blend: 20% Poisson model + 80% ranking
6. Результат: реалистичные вероятности
```

### Для клубных матчей:
```
1. Используем стандартную Poisson модель
2. Blend: 60% model + 40% market (если есть кф)
3. Home advantage: 1.08-1.15
```
