# 📊 Post-Match Analysis Reports - Новая функция!

## Что нового в v2.0

### 🎯 Автоматические отчеты после матчей

Теперь бот **автоматически отслеживает** матчи, на которые вы запрашивали аналитику, и **присылает отчет** после завершения матча!

---

## 📋 Как это работает

### 1. Вы запрашиваете анализ матча
```
Вы: /start → 📅 Анализ матча → ⚽ Футбол → Сегодня (4) → 
🌍 Квалификация ЧМ (Европа) (4) → Sweden vs Poland
```

### 2. Бот сохраняет прогноз
Бот запоминает:
- 📊 Вероятности 1X2 (П1: 41%, X: 23%, П2: 36%)
- ⚽ Тоталы (ТБ 2.5: 65%, ТМ 2.5: 35%)
- 🎯 Обе забьют (Да: 66%, Нет: 34%)
- 🎲 Вероятные счета (1:1, 2:1, 1:2)

### 3. Матч завершается
Sweden 2:1 Poland (FT)

### 4. Бот присылает отчет
```
⚽ РЕЗУЛЬТАТ МАТЧА
Sweden 2:1 Poland
🏆 World Cup - Qualification Europe

📊 ИСХОД 1X2:
✅ П1 (прогноз: 41%)
   Фактически: П1 (2:1)

⚽ ТОТАЛ ГОЛОВ:
✅ ТБ 2.5 (прогноз: 65%)
   Фактически: 3 гола

🎯 ОБЕ ЗАБЬЮТ:
✅ Да (прогноз: 66%)
   Фактически: Да

🎲 ТОЧНЫЙ СЧЕТ:
❌ Прогноз: 1:1 (12%)
   Фактически: 2:1

📈 ИТОГ:
✅ Отличный прогноз! Точность: 100%

⚠️ Ставки — это риск. Прошлые результаты не гарантируют будущие.
```

---

## ⚙️ Технические детали

### Отслеживаемые данные
- **Match ID** — для проверки результата
- **User ID** — чтобы знать, куда отправить отчет
- **Прогнозы** — 1X2, тоталы, BTTS, точный счет
- **Дата матча** — для проверки завершения

### Периодичность проверки
- **Каждые 30 минут** бот проверяет статус tracked матчей
- Если матч завершен (FT, AET, PEN) — отправляется отчет
- Через **7 дней** tracked матчи автоматически удаляются

### Хранение данных
- Данные сохраняются в `tracked_matches.json`
- Формат: JSON
- Автоочистка старых записей

## 🔧 Управление

### Посмотреть tracked матчи
```python
from match_results_tracker import MatchResultsTracker

tracker = MatchResultsTracker()
print(f"Tracked matches: {len(tracker.tracked_matches)}")

for match_id, match in tracker.tracked_matches.items():
    print(f"  • {match.home_team} vs {match.away_team} (Report: {match.report_sent})")
```

### Очистить tracked матчи
```python
tracker.cleanup_old_matches(days=0)  # Удалить все
```

## 📊 Статистика

Бот отправляет отчеты только если:
- ✅ Матч завершен (статус FT, AET, PEN)
- ✅ Пользователь запрашивал анализ
- ✅ Отчет ещё не был отправлен
- ✅ Прошло не более 7 дней
