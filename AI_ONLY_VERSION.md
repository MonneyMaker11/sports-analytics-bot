## 🧮 Как Claude рассчитывает вероятности:

**ШАГ 1: БАЗА — рыночные коэффициенты**
```
Если кф есть в API:
• Босния: 7.50 → 13.3%
• Ничья: 4.25 → 23.5%
• Италия: 1.52 → 65.8%
• После удаления маржи: 13% | 23% | 64%
```

**ШАГ 2: Корректировка по форме**
```
• 5 побед подряд: +8%
• 4 победы: +5%
• 3 победы: +2%
• 2 победы: 0%
• 1 победа: -3%
• 0 побед: -5%
```

**ШАГ 3: Учёт xG**
```
• xG > 2.0: элитная атака (+5%)
• xG 1.5-2.0: хорошая атака (+2%)
• xG 1.0-1.5: средняя (-2%)
• xG < 1.0: слабая (-5%)
```

**ШАГ 4: H2H**
```
• 4+ победы подряд: +7%
• 3 победы подряд: +5%
• 2 победы подряд: +3%
• Паритет: 0%
```

**ШАГ 5: Домашнее преимущество**
```
• Клубы: +10-12%
• Сборные: +6-8%
```

**ШАГ 6: Травмы**
```
• Ключевой игрок вне: -12%
• 2+ ключевых: -20%
```

**ШАГ 7: Мотивация**
```
• Критический матч: +5%
• Обычный: 0%
• Без мотивации: -5%
```

**ШАГ 8: ФИНАЛЬНЫЙ РАСЧЁТ**
```
Сложи все корректировки с базой.
Проверь: П1 + X + П2 = 100%
```
