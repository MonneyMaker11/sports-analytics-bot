## ✨ Что нового в v3.0

### 🎯 Главная функция: Post-Match Reports

**Автоматические отчеты после матчей!**

- Бот запоминает прогнозы на матчи, которые вы запрашиваете
- После завершения матча присылает отчет с точностью прогноза
- Проверка каждые **5 минут** после окончания матча
- Отчеты приходят **только вам** (персонально для каждого пользователя)

## 📦 Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/MonneyMaker11/sports-analytics-bot.git
cd sports-analytics-bot

# Установить зависимости
pip install -r requirements.txt

# Настроить переменные окружения
cp .env.example .env
# Отредактировать .env (добавить токены)

# Запустить бота
python main.py
```

## 🎯 Как использовать Post-Match Reports

1. **Запросите анализ матча:**
   ```
   /start → 📅 Анализ матча → ⚽ Футбол → 
   Выберите дату → Выберите лигу → Выберите матч
   ```

2. **Дождитесь окончания матча**

3. **Получите отчет через 5 минут:**
   ```
   ⚽ РЕЗУЛЬТАТ МАТЧА
   Sweden 2:1 Poland
   
   📊 ИСХОД 1X2:
   ✅ П1 (прогноз: 41%)
      Фактически: П1
   
   📈 ИТОГ:
   ✅ Отличный прогноз! Точность: 100%
   ```
