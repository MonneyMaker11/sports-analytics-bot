## Как использовать

### 1. Запуск бота

```bash
cd /Users/ilyailyx/Desktop/sports-analytics-bot
source venv/bin/activate
python main.py
```

### 2. В Telegram

1. Откройте бота
2. Нажмите **"📅 Анализ матча"**
3. Выберите **"⚽ Футбол"**
4. Выберите **"Сегодня (4)"** ← покажет количество матчей квалификации
5. Выберите **"🌍 Квалификация ЧМ (Европа) (4)"**
6. Выберите матч для анализа
