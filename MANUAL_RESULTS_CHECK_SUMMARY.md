## 📝 Заметки

### tracked_matches.json

- Сохраняется в корне проекта
- Формат: JSON
- Структура:
  ```json
  {
    "1537581": {
      "match_id": "1537581",
      "user_id": 379167016,
      "home_team": "Sweden",
      "away_team": "Poland",
      "predictions": {...},
      "report_sent": false
    }
  }
  ```

### Очистка старых матчей:

- Автоматически через 7 дней
- Метод: `tracker.cleanup_old_matches(days=7)`
- Можно вызвать вручную
