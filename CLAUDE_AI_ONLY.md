## 📋 Новый промт для Claude

### Ключевые изменения:

**БЫЛО:**
```
Ты — профессиональный спортивный аналитик...

```

**СТАЛО:**
```
🎯 PROMPT: Спортивный аналитик ставок (Football Betting Analyst)

Ты — профессиональный аналитик футбольных матчей с уклоном в беттинг.

📊 ОСНОВНЫЕ ПРИНЦИПЫ:
• Используй только реальные данные из предоставленной статистики
• Ты САМ рассчитываешь вероятности на основе данных

🧮 КАК РАССЧИТЫВАТЬ ВЕРОЯТНОСТИ:
1. Оцени силу команд по форме (последние 5 матчей)
2. Учти xG (ожидаемые голы) - кто больше создаёт
3. Посмотри H2H (личные встречи)
4. Проверь коэффициенты букмекеров (рыночная оценка)
5. Рассчитай проценты так, чтобы сумма = 100%

Пример:
- Команда A: форма 4/5 побед, xG 1.9, кф 1.52 → ~65%
- Команда B: форма 2/5 побед, xG 1.4, кф 7.5 → ~17%
- Ничья: ~18%
```

## 🔄 Поток данных

### 1. main.py получает данные
```python
match_details = self.parser.get_match_details(match_id)
# - home_team_form
# - away_team_form
# - h2h_matches
# - statistics (xG, shots, etc.)
# - odds (bookmakers)
# - standings
```

### 2. main.py передаёт Claude
```python
analysis = await self.analyzer.generate_analysis(match_details)
# Никаких предварительных расчетов!
# Claude получает ВСЕ raw данные
```

### 3. Claude анализирует
```python
# ai_analyzer.py - _build_prompt()
data = f"""
⚽ Босния vs Италия | World Cup - Qualification Europe

📈 ФОРМА:
Босния: WWWWL (12–5)
Италия: LWWWW (14–6)

⚙️ ПРОДВИНУТАЯ СТАТИСТИКА:
• xG: 1.85 vs 1.92
• Стиль: attacking vs very_attacking

📊 ВЕРОЯТНОСТИ (данные для анализа):
[Claude САМ рассчитывает]

💰 КОЭФФИЦИЕНТЫ:
Босния: 7.50 | Ничья: 4.25 | Италия: 1.52
"""
```

### 4. Claude выдаёт прогноз
```

📊 ВЕРОЯТНОСТИ

Босния: W-W-W-W-L (12-5)
Италия: L-W-W-W-W (14-6)

🎯 H2H
• Италия выиграла 4 из 5 последних встреч

📌 КЛЮЧЕВЫЕ ФАКТОРЫ

🎯 РЕКОМЕНДАЦИИ
```
